# ShebaBD Frontend Client

A responsive React + Vite + TypeScript + Tailwind CSS client interface for the ShebaBD Doctor Appointment System.

## Features
- Interactive Specialties category lookup
- Doctor explore directory with advanced sorting, search, and divisions filtering
- Chambers schedules view and patient booking forms
- Client panel displaying reservations and daily charts using Recharts
- Quick Demo Login presets for Patients, Doctors, and Administrators
- Fully responsive across desktop, tablet, and mobile screens

## Prerequisites
- Node.js (LTS version)
- NPM

## Installation & Setup
1. Install frontend dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
   *Client will be active at [http://localhost:3000](http://localhost:3000).*
